# Aging Job Prospects Project
This assignment we were challegned to make a dashboard analyzing and vizualizing
trends or patterns within the job market between 2011-2024. We were tasked to each
make a graph relating to how factors such as race, gender, and age all play
a role in employment as well as how they change over the years. This challenge
also required that we use Github in order to track changes and collaborate with our
group effectively using commits and pusing/pulling updates.This is why our team decided 
to answer the question How do factors like age, race, gender, and time affect 
employment?

#How to Access
Our group began by making an Rmd using shiny which was titled AgingJobUpdate.
However we wanted to create a more interactive eye-catching look so we used a .R file
titled Download.R where you can access the interactive dashboard containing an about/overivew
section, all 7 of our group members graphs, as well as data analysis summary describing patterns
and trends noted within the graphs. The top panels allow you to click through the different
sections and graphs and they are all labeled relating to what the graph is analyzing.
